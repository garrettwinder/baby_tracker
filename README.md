# baby_tracker

Track basic baby activity (pee, poo, eat, sleep).

## This train runs on Rails (toot, toot!)

- First timer?
  - `bundle install`
  - `rails db:migrate`
- Repeat visitor?
  - `bin/dev`
  - Go to: http://localhost:3000
